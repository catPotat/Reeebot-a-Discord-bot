A Discord bot named *Reeebot*


### Hobbyist project bot features:
- LLM Chat bot
- Avatar recognizer (no more)
- Quick image edits
- pixiv search
- discord jupyter notebook because

```

flyctl secrets set KEY1=VALUE1 KEY2=VALUE2
```
